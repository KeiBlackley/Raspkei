About

Raspkei is a Raspberry Pi scripting tool for automation, designed to simplify tasks and improve efficiency. Mainly coded in Python3, with shell, batch, and custom programs for automation and user experience.

This project is Open-Source as long as credentials of the Original Developer (@KeiBlackley) or (Keirran Blackley) are mentioned and the license is copied over.

Getting Started

Hardware

  • Recommended: Raspberry Pi 3B+ (tested on multiple models)
  • Optional: Telstra 4G MF833v Modem, Waveshare UPS HAT

Software

  • Raspberry Pi OS (use Raspberry Pi Imager, enable SSH, set hostname to raspkei)
  • Dependencies: PHP8, MariaDB, Apache2, git, hostapd, dnsmasq, iptables

Installation

git clone https://github.com/KeiBlackley/Raspkei
    cd Raspkei
    sudo chmod +x run.sh
    ./run.sh

Quick Start

usage: raspkei [options]
    options:
        --apache required_boolean [on/off]   Installs Apache Web Server
        --hotspot required_boolean [on/off]  Sets up WiFi Access Point

Project Structure

Raspkei
        -> web      # Web Server Files
            -> runapache, resetapache
        -> wifi     # Hotspot Files
            -> runhotspot
        -> win      # Windows Files

Technical Details

  • Web Server: Apache, MariaDB, PHP
  • Hotspot: RaspAP, Optional: nodogsplash, VPN, ADGuard DNS
